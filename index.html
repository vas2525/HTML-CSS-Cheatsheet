HTML & CSS Cheatsheet

A quick reference guide to commonly used HTML elements and CSS selectors/properties.

HTML Basics

Tag Name Description
<!DOCTYPE> Document Type Defines the document type
<html> HTML Element Defines an HTML document
<head> Head Element Contains metadata/information for the document
<title> Title Defines a title for the document
<body> Body Defines the document's body
<h1> to <h6> Headings Defines HTML headings
<p> Paragraph Defines a paragraph
<br> Line Break Inserts a single line break
<hr> Thematic Break Defines a thematic change in the content
<!--...--> Comment Defines a comment

Formatting

Defines contact information for the author/owner of a document/article
Tag Name Description
<acronym> Acronym Not supported in HTML5. Use <abbr> instead.
Defines an acronym
<abbr> Abbreviation or an Acronym Defines an abbreviation or an acronym
<address> Address
<b> Bold Defines bold text
<bdi> Bidirectional Isolation Isolates a part of text that might be formatted in a different direction from other text outside it
<bdo> Bidirectional Override Overrides the current text direction
<big> Big Text Not supported in HTML5. Use CSS instead. Defines big text
<blockquote> Block Quote Defines a section that is quoted from another source
<center> Center Not supported in HTML5. Use CSS instead. Defines centered text
<cite> Citation Defines the title of a work
<code> Code Defines a piece of computer code
<del> Deleted Text Defines text that has been deleted from a document
<dfn> Definition Specifies a term that is going to be defined within the content
<em> Emphasis Defines emphasized text
<font> Font Not supported in HTML5. Use CSS instead. Defines font, color, and size for text
<i> Italic Defines a part of text in an alternate voice or mood
<ins> Inserted Text Defines a text that has been inserted into a document
<kbd> Keyboard Input Defines keyboard input
<mark> Marked Text Defines marked/highlighted text
<meter> Meter Defines a scalar measurement within a known range (a gauge)
<pre> Preformatted Text Defines preformatted text
<progress> Progress Represents the progress of a task
<q> Quotation Defines a short quotation
<rp> Ruby Parenthesis Defines what to show in browsers that do not support ruby annotations
<rt> Ruby Text Defines an explanation/pronunciation of characters (for East Asian typography)
<ruby> Ruby Annotation Defines a ruby annotation (for East Asian typography)
<s> Strikethrough Defines text that is no longer correct
<samp> Sample Output Defines sample output from a computer program
<small> Small Text Defines smaller text
<strike> Strikethrough Not supported in HTML5. Use <del> or <s> instead. Defines strikethrough text
<strong> Strong Defines important text
<sub> Subscript Defines subscripted text
<sup> Superscript Defines superscripted text
<template> Template Defines a container for content that should be hidden when the page loads
<time> Time Defines a specific time (or datetime)
<tt> Teletype Text Not supported in HTML5. Use CSS instead. Defines teletype text
<u> Underline Defines some text that is unarticulated and styled differently from normal text
<var> Variable Defines a variable
<wbr> Word Break Defines a possible line-break
